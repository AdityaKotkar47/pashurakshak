PashuRakshak API Service

This is the API service for the PashuRakshak animal rescue application.

Mobile App API Endpoints:

Test Connection:

GET /api/volunteers/test

Volunteer Endpoints:

POST /api/volunteers/m-login - Login
GET /api/volunteers/m-profile - Get profile
GET /api/volunteers/m-missions - Get missions
PUT /api/volunteers/m-missions/:id/status - Update mission status
POST /api/volunteers/m-missions/:id/notes - Add notes

Rescue Request Endpoints:

GET /api/rescue/m-requests/user/:userId - Get user's requests
GET /api/rescue/m-requests/:id/timeline - Get rescue timeline
