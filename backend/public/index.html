PashuRakshak API

API Status

This is the base API for the PashuRakshak application. The API provides endpoints for animal rescue operations.

GET /api/volunteers/public-test

Mobile-friendly Endpoints

These endpoints are optimized for mobile applications with improved security handling:

  • POST /api/volunteers/m-login
  • GET /api/volunteers/m-profile
  • GET /api/volunteers/m-missions

Debug Endpoints

These endpoints can be used to troubleshoot authentication issues:

  • GET /api/volunteers/token-info
  • GET /api/volunteers/verify-token
